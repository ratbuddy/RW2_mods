# Quicksave

F5/F8 quicksave and quickload for Rift Wizard 2.

## Usage

- **F5** — Quicksave (only when it's your turn)
- **F8** — Quickload (anytime, including after death, game over screens, and the title menu)

One save slot — each F5 overwrites the previous quicksave. Starting a new game deletes the quicksave so it can't carry over between runs.

## Install

Drop the `quicksave` folder into your `mods/` directory.
